# QuizQuest
QuizQuest is a gamified learning prototype using Unity 2D. Players engage with quiz challenges enhanced by animations and feedback for fun learning.


# 项目文件夹结构说明

该项目的文件夹结构旨在帮助团队成员快速找到相关资源，并提高项目的可维护性。以下是项目的文件夹组织和每个文件夹的详细描述：

## /Assets
所有游戏资源都集中存放在此文件夹下。它包括了UI设计、动画、共享资源、场景、脚本等内容。

### /UI
存放所有与UI（用户界面）相关的资源和文件。
- **/Scripts**：包含所有处理UI逻辑的脚本。比如按钮点击、题目展示、答案选择等。
- **/Prefabs**：存放UI组件的预制件。例如，按钮、文本框、进度条等UI元素的预设文件。
- **/Animations**：UI相关的动画文件。例如，按钮点击动画、界面切换动画等。
- **/Sprites**：存放UI所需的图片资源。包括背景图、按钮图标等视觉资源。
- **/Fonts**：UI字体文件。此文件夹包含所有游戏界面使用的字体资源。
- **/Shaders**：存放UI所需的Shader文件。包含各种用于UI效果的Shader，例如渐变、阴影、特效等。

### /Animations
该文件夹专门存放动画相关资源。包括火柴人动画等其他角色动画。
- **/Scripts**：控制火柴人动画的脚本。用于控制动画的播放、切换等。
- **/Prefabs**：存放动画角色的预制件。包括所有动画的角色模型及其绑定的动画。
- **/Animator**：Animator控制器文件。用于管理火柴人角色的不同动画状态（如闲置、跳跃、跑步等）。
- **/Textures**：动画所需的纹理素材。包括火柴人的材质、贴图等。
- **/Audio**：与动画相关的音效文件。例如，跳跃声、摔倒声等。

### /Shared
存放UI和动画模块共享的资源。
- **/Sprites**：共享的图片资源。例如，游戏背景图、通用图标等。
- **/Sounds**：共享的音效资源。例如，按钮点击音效、界面切换音效等。
- **/Shaders**：共享的Shader文件。例如，游戏通用的Shader效果等。

### /Scenes
存放游戏的场景文件。
- **MainScene**：主要的游戏场景文件，包含游戏的核心内容和UI界面等。

### /Resources
存放其他资源文件。
- **/Localization**：本地化资源，包含翻译文件等，用于支持不同语言版本的游戏。
- **/Fonts**：游戏中使用的字体文件。
- **/Audio**：游戏音效资源，包含背景音乐、音效等。

### /Scripts
存放项目中的所有脚本文件。
- **/Managers**：管理游戏流程和逻辑的脚本。例如，游戏控制脚本、状态管理脚本、计分系统等。
- **/Utilities**：一些辅助工具类脚本，包含通用的功能脚本，比如数据模型、工具函数等。

### /Documentation
存放项目的文档文件。
- **/DesignDocs**：设计文档，包含UI设计、动画设计、功能实现等详细说明。
- **/TechnicalDocs**：技术文档，包含项目架构说明、接口文档、开发流程等技术相关内容。

---

通过这种文件夹结构，项目团队可以更加清晰地组织和管理资源，提高开发效率。每个模块的功能和文件都被明确划分，确保团队成员能够快速找到所需的资源和文件。
